# Student_Management 사용법

- 학생, 교수, 관리자로 나눠저있는 학사관리 프로그램
- 학생 : 학생정보 수정, 점수확인, 강의 등록 등이 가능함
- 교수 : 교수정보 수정, 과목, 강의 생성 및 점수 수정이 가능
- 관리자 : 학생과 교수의 계정을 관리한다. 아이디 : 00 비밀번호 : 00


<img src="https://user-images.githubusercontent.com/47252346/128630682-9763a507-834b-403c-a467-7d47633f187f.png"  width="700" height="500"/>


* 로그인 / 회원가입
  - 로그인 : 회원가입한 아이디, 비밀번호 입력
  - 회원가입 : 학생과 교수 중 하나를 선택하고 아이디, 비밀번호 입력, 교수와 학생 아이디 중복 불가 <br>회원가입한 아이디가 학생, 교수코드가 됨


### * 학생으로 로그인할 경우


![image](https://user-images.githubusercontent.com/47252346/148680196-2dd12c34-0f62-4836-840b-03987d968de5.png)


#### - 학생 정보 
   - 기본적으로 학생코드와 비밀번호만 저장되있음.
   - 내 정보 저장하기 : 텍스트 상자에 입력된 정보를 데이터베이스에 저장한다.
   - 내 정보 불러오기 : 텍스트 상자에 입력된 정보를 데이터베이스에 불러온다.

![image](https://user-images.githubusercontent.com/47252346/148680150-c5fbebe3-d55d-4d94-8eb2-45647a206051.png)

![image](https://user-images.githubusercontent.com/47252346/129476189-d05eb5d6-87c9-46da-af53-38a16eb55684.png)


#### - 성적
   - 점수 확인하기 : 해당 학생이 등록한 강의의 점수를 띄움. 교수가 성적을 입력하며 입력되지 않았을 경우 미입력으로 표기됨
   - 시간표 보기 : 해당 학생이 등록한 강의의 시간표를 띄우고 강의 시작 시간으로 정렬함
  
   
   
   ![image](https://user-images.githubusercontent.com/47252346/128630899-ebcf9767-aab3-4e2e-afa9-e535596afa34.png)

   
#### - 강의
   - 강의 등록 : 클릭한 리스트의 강의를 수강신청 한다.
   - 강의 취소 : 등록한 강의의 수강신청을 취소한다.

### * 교수로 로그인할 경우

![image](https://user-images.githubusercontent.com/47252346/148680203-cbd61d38-e89e-4841-ac66-d25a83a72a63.png)



#### - 교수 정보
 - 기본적으로 교수코드만 저장되있음. 정보 입력후 저장시 데이터베이스에 반영하고 불러오기 클릭시 반영된 정보를 입력칸에 표시함
  
  ![image](https://user-images.githubusercontent.com/47252346/129476290-c0ef8328-92cb-420d-b485-4f13ca0c82ad.png)

  
  
#### - 과목
   - 과목 생성 : 텍스트 박스에 입력된 대로 과목을 생성한다. 같은 과목코드 사용 불가<br>
   - 과목 제거 : 리스트에서 클릭된 과목을 삭제한다.<br>
   - 과목 수정 : 리스트에서 클릭된 과목의 정보를 텍스트 박스에 입력된 대로 수정한다.<br>
  
  ![image](https://user-images.githubusercontent.com/47252346/129476692-af93e41c-af45-41ec-af5a-651056b36448.png)

  
  
#### - 강의 
   - 강의 생성 : 텍스트 박스에 입력된 대로 과목을 생성한다. 같은 강의코드 사용 불가
   - 강의 제거 : 리스트에서 클릭된 강의을 삭제한다.
   - 강의 수정 : 리스트에서 클릭된 강의의 정보를 텍스트 박스에 입력된 대로 수정한다.

 ![image](https://user-images.githubusercontent.com/47252346/129476711-d5cae1ff-840b-4424-b2a9-f4318fa81aff.png)
 
  ##### - 수강자 확인 : 리스트에서 클릭된 강의를 수강하는 학생들을 표시한다.
    - 성적 조회 : 데이터베이스에 등록된 학생들의 성적을 불러온다
    - 성적 수정 : 학생들의 성적을 입력된 값으로 변경한다
    - 학생 수강 취소 : 클릭한 학생의 강의 수강을 취소한다
